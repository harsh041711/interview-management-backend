## How to capture

- Resolution: ≥ 1280px wide preferred so text is readable on GitHub.
- Format: PNG (or JPG for very large screenshots).
- Hide any real candidate names / emails if these will be shared publicly — use the seeded test data instead.
- After dropping the files in this folder, just push to `dev` and they'll appear in `FEATURES.md` on GitHub automatically.
